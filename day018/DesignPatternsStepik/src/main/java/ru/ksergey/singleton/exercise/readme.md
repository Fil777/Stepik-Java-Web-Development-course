## Задание: "Система логирования для интернет-магазина"

**Условие:**
Создайте систему логирования для интернет-магазина, используя паттерн Singleton. Система должна записывать все важные
события в приложении.

**Требования:**

1. Создать класс `Logger` с использованием паттерна Singleton
2. Реализовать следующие методы:
    - `logError(String message)` - для логирования ошибок
    - `logInfo(String message)` - для логирования информационных сообщений
    - `logWarning(String message)` - для логирования предупреждений
    - `getLogHistory()` - для получения истории логов

**Проверка работы:**

```java
public class Shop {
    public static void main(String[] args) {
        // Получить экземпляр логгера
        // Залогировать несколько событий
        // Проверить историю логов
    }
}
```

## Дополнительные задания:

1. Добавьте временные метки к логам
2. Реализуйте уровни логирования (DEBUG, INFO, WARNING, ERROR)
3. Добавьте возможность фильтрации логов по уровню
4. Реализуйте сохранение логов в файл
5. Добавьте ограничение на размер истории логов (например, последние 100 записей)

## Критерии оценки:

- Правильность реализации паттерна Singleton
- Корректная работа всех методов
- Обработка краевых случаев
- Качество и читаемость кода
