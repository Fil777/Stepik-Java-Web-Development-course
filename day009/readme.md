$$
F(n) = 
\begin{cases}
0, & \text{если } n = 0 \\
1, & \text{если } n = 1 \\
F(n-1) + F(n-2), & \text{если } n > 1
\end{cases}
$$

8. Напишите метод, который принимает число и возвращает true, если оно простое, и false, если составное. Число является простым, если оно делится только на 1 и на само себя.

$\forall a \in \mathbb{N}, 1 < a < p: p \nmid a$, где: $p$ - число, которое проверяется на простоту, $a$ - любое натуральное число в диапазоне от 2 до $p-1$, $\nmid$ - символ, обозначающий "не делится нацело"

Эта формула означает, что число $p$ является простым, если оно не делится нацело ни на одно натуральное число $a$ в диапазоне от 2 до $p-1$.
[решение](./exercises/Exercise08.java)

## Техническое задание на доработку проекта "Система управления библиотекой"

### Цель доработки
Реструктуризация кода для улучшения его организации, читаемости и масштабируемости путем разделения логики на отдельные методы.

Итоговая структура приложения:

```text
core/
    App.java
    BookManager.java
    Program.java
    Terminal.java
```

### Основные задачи

1. **Создание пакета**
   - Создайте пакет `core` в рамках которого будет реализовываться доработка приложения.

2. **Класс вывода в консоль**
   - Опишите класс `Terminal`, в котором будет два метода-обёртки
   - `write()` над `System.out.print()`
   - `writeLine()` над `System.out.println()`

3. **Создание класса BookManager**
   - Перенести всю логику управления библиотекой в отдельный класс `BookManager` в пакете `core`.
   - Реализовать поля для хранения книг и пользователей.

4. **Методы управления книгами**
   - Создать метод `addBook()` для добавления новой книги.
   - Реализовать метод `removeBook()` для удаления книги.
   - Добавить метод `searchBook()` для поиска книги по названию.
   - Разработать метод `displayAllBooks()` для отображения всех книг.

5. **Методы управления пользователями**
   - Создать метод `addUser()` для добавления нового пользователя.
   - Реализовать метод `removeUser()` для удаления пользователя.
   - Добавить метод `displayUserBooks()` для отображения книг конкретного пользователя.

6. **Методы для операций с книгами**
   - Реализовать метод `borrowBook()` для выдачи книги пользователю.
   - Создать метод `returnBook()` для возврата книги в библиотеку.

7. **Вспомогательные методы**
   - Добавить метод `isBookAvailable()` для проверки доступности книги.
   - Реализовать метод `getUserByName()` для поиска пользователя по имени. (можно не описывать, в видео расскажу почему)

8. **Обработка ошибок**
   - Реализовать проверку наличия книги перед её удалением или выдачей.
   - Добавить проверку существования пользователя перед операциями с его книгами.

9. **Начальная инициализация проекта**
   - Создайте метод `initializeLibrary()` в котором будет логика стартового наполнения книг и пользователей.
   
10. **Главный класс приложения**
    - Создать отдельный класс `App` с методом `run()`.
    - Реализовать в нем интерфейс пользователя и вызовы методов класса BookManager.

11. **Запуск приложения**
    - Создайте класс `Program` внутри которого будет точка входа `main()` и вызовите метод `App.run()`.Протестируйте работу приложения.
