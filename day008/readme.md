## Упражнения на отработку методов

### Часть 1

1. Напишите метод для умножения трех чисел.
2. Напишите метод для вычисления количества минут в указанном количестве дней.  
3. Напишите метод, который принимает N минут и возвращает количество целых часов без минут.  
4. Напишите метод для нахождения объема куба с заданной длиной ребра `edge`.  
5. Напишите метод, который вычисляет площадь треугольника по формуле Герона с заданными сторонами $a$, $b$, и $c$.  

$$S = \sqrt{s(s-a)(s-b)(s-c)}$$
, где $$s = \frac{a+b+c}{2}$$ - полупериметр треугольника.

6. Напишите метод, который вычисляет длину окружности с заданным радиусом.  
$$C = 2\pi r$$
, где $r$ - радиус окружности.

7. Напишите метод для перевода градусов Фаренгейта в градусы Цельсия.  
$$C = \frac{5}{9}(F - 32)$$
, где $F$ - температура в градусах Фаренгейта, $C$ - температура в градусах Цельсия.

8. Напишите метод для вычисления скалярного произведения двух векторов в трёхмерном пространстве.
$$\vec{a} \cdot \vec{b} = a_1b_1 + a_2b_2 + a_3b_3$$
, где $\vec{a} = (a_1, a_2, a_3)$ и $\vec{b} = (b_1, b_2, b_3)$ - два вектора в трёхмерном пространстве.

9. Напишите метод для вычисления значения выражения $$ax^2 + bx + c$$
, где $a$, $b$, $c$ генерируются случайно, а $x$ - входной параметр метода.   
$$ax^2 + bx + c$$
, где $a$, $b$, $c$ - коэффициенты, $x$ - переменная.

10. Напишите метод для вычисления расстояния, пройденного объектом, движущимся с начальной скоростью $v_0$, ускорением $a$ за время $t$.   
$$S = v_0t + \frac{1}{2}at^2$$
, где $S$ - расстояние, $v_0$ - начальная скорость, $a$ - ускорение, $t$ - время.

11. Напишите метод, который принимает число и возвращает true, если оно четное, и false, если нечетное.  
12. Напишите метод для расчета стоимости проезда в зависимости от дистанции: до 100 км — тариф А, свыше 100 км — тариф Б.  
13. Напишите метод для определения сезона года по номеру месяца.  
14. Напишите метод, который в зависимости от выбранного режима работы (легкая, средняя, тяжелая физическая активность) возвращает оптимальную температуру для комфортной работы: 20-22°C для лёгкой, 18-20°C для средней, 16-18°C для тяжелой.
15. Напишите метод, который принимает число и возвращает строку, указывающую диапазон числа: маленькое (0-10), среднее (11-100), большое (101 и выше).
