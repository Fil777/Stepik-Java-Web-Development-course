# Упражнения

## Задача 1: Калькулятор индекса массы тела (ИМТ)

Создайте программу, которая рассчитывает индекс массы тела (ИМТ) на основе введенных пользователем данных о весе (в килограммах) и росте (в метрах).

**Формула расчета ИМТ:**
$$ИМТ = \frac{вес}{рост^2}$$

После расчета, программа должна вывести ИМТ и категории веса согласно следующей шкале:
- Меньше 18.5 - Недостаточный вес
- 18.5 - 24.9 - Нормальный вес
- 25 - 29.9 - Избыточный вес
- 30 и более - Ожирение

Пример:
```text
Введите ваш вес в килограммах: 90
Введите ваш рост в метрах: 1,85
Ваш индекс массы тела (ИМТ): 26,30
Меньше 18.5 - Недостаточный вес
18.5 - 24.9 - Нормальный вес
25 - 29.9 - Избыточный вес
30 и более - Ожирение
```

## Задача 2: Калькулятор площади поверхности и объема цилиндра

Напишите программу, которая запрашивает у пользователя радиус основания и высоту цилиндра, а затем вычисляет и выводит его площадь поверхности и объем.

**Формулы:**
- Площадь поверхности: $$S = 2\pi r^2 + 2\pi rh = 2\pi r (r + h)$$ 
- Объем: $$V = \pi r^2h$$

Используйте `Math.PI` для значения π и округлите результаты до двух знаков после запятой.

Тесты: 
- [Объема цилиндра](https://calc.by/math-calculators/calculator-volume-cylinder.html)
- [Площади поверхности цилиндра](https://ru.onlinemschool.com/math/assistance/figures_area_1/cylinder)

Пример:
```text
Введите радиус основания цилиндра: 1
Введите высоту цилиндра: 1
Площадь поверхности цилиндра: 12,57
Объем цилиндра: 3,14
```

## Задача 3: Калькулятор расстояния между двумя точками на плоскости

Напишите программу, которая запрашивает у пользователя координаты двух точек (x1, y1) и (x2, y2) на плоскости, а затем вычисляет и выводит расстояние между ними.

**Формула:**
$$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$

Используйте `Math.sqrt()` для вычисления квадратного корня и округлите результат до двух знаков после запятой.

Пример:
```text
Введите координату x1: 0
Введите координату y1: 0
Введите координату x2: 1
Введите координату y2: 1
Расстояние между (0,00, 0,00) и (1,00, 1,00) = 1,41
```

## Техническое задание (ТЗ) на доработку программы "Калькулятор чаевых"

### **Цель доработки**

Модифицировать существующее консольное Java-приложение "Калькулятор чаевых" для обеспечения интерактивного ввода данных пользователем.

### **Функциональные требования**

1. **Ввод данных:**
   - Реализовать интерактивный ввод следующих параметров:
     - **Сумма счёта (`billAmount`):** 
       - Запрос: "Введите сумму счёта: "
       - Тип данных: `double`
     - **Процент чаевых (`tipPercentage`):**
       - Запрос: "Введите процент чаевых: "
       - Тип данных: `double`
     - **Количество участников (`numberOfPeople`):**
       - Запрос: "Введите количество участников: "
       - Тип данных: `int`

### **Нефункциональные требования**

- Использовать класс `Scanner` для считывания пользовательского ввода
- Сохранить существующую логику расчётов и форматированного вывода результатов

### **Пример работы после доработки**

```text
Введите сумму счёта: 1000
Введите процент чаевых: 10
Введите количество участников: 4

Сумма счёта: 1000,00
Сколько человек участвовало в обеде: 4
Общий счёт: 1100,00
Процент чаевых: 10.0%
С каждого: 275,00
```
