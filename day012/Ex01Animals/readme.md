# Это наследственный зоопарк

Напишите программу на Java, которая описывает классы Dog, Cat и Bird, вызывает их методы и выводит результаты в консоль.

1. Создайте класс Dog со следующими полями:
   - name (имя) типа String
   - age (возраст) типа int
   - weight (вес) типа double
   - breed (порода) типа String

2. Добавьте в класс Dog следующие методы:
   - конструктор, принимающий параметры name, age, weight и breed
   - void bark() - выводит в консоль "{имя} лает: Гав! Гав!"
   - void fetch() - выводит в консоль "{имя} приносит мяч"

3. Создайте класс Cat со следующими полями:
   - name (имя) типа String
   - age (возраст) типа int
   - weight (вес) типа double
   - furColor (цвет шерсти) типа String

4. Добавьте в класс Cat следующие методы:
   - конструктор, принимающий параметры name, age, weight и furColor
   - void meow() - выводит в консоль "{имя} мяукает: Мяу!"
   - void scratch() - выводит в консоль "{имя} царапает мебель"

5. Создайте класс Bird со следующими полями:
   - name (имя) типа String
   - age (возраст) типа int
   - weight (вес) типа double
   - species (вид) типа String
   - canFly (умеет летать) типа boolean

6. Добавьте в класс Bird следующие методы:
   - конструктор, принимающий параметры name, age, weight, species и canFly
   - void chirp() - выводит в консоль "{имя} чирикает: Чик-чирик!"
   - void fly() - если canFly равно true, выводит в консоль "{имя} летает", иначе выводит "{имя} не может летать"

7. Запустите программу и проверьте вывод в консоли.

Пример вывода:
```
Бобик лает: Гав! Гав!
Бобик приносит мяч
Мурзик мяукает: Мяу!
Мурзик царапает мебель
Кеша чирикает: Чик-чирик!
Кеша летает
```
