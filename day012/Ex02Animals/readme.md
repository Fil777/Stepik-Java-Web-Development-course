## Наследование

Наследование - это механизм в объектно-ориентированном программировании, который позволяет создавать новые классы на основе существующих. В Java наследование реализуется с помощью ключевого слова `extends`.

Важно знать:

1. Родительский класс (суперкласс) передает свои поля и методы дочернему классу (подклассу).
2. Подкласс может добавлять новые поля и методы, а также переопределять методы суперкласса.
3. Ключевое слово `super` используется для вызова конструктора и методов суперкласса.
4. Java поддерживает только одиночное наследование классов, но множественное наследование интерфейсов.
5. Наследование способствует повторному использованию кода и созданию иерархии классов.
6. Модификатор `final` предотвращает наследование класса или переопределение метода.

Наследование - инструмент для создания гибких и расширяемых программ, позволяющий организовывать код в логические иерархии и избегать дублирования.
