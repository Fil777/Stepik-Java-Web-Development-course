# Техническое задание "Виртуальный зоопарк"

Цель: Разработать программу, которая создает и управляет виртуальным зоопарком с различными животными.

Требования:

1. Определите логику, которая:
   - Инициализирует общее количество животных (20)
   - Случайным образом распределяет животных между собаками, кошками и птицами
   - Создаст коллекцию для каждого типа животных
   - Заполнит коллекции, создавая экземпляры животных со случайными параметрами
   - Выведет информацию о количестве животных каждого вида
   - Продемонстрирует действия каждого животного, вызывая их методы

2. Используйте класс Random для генерации случайных чисел:
   - Для распределения количества животных
   - Для генерации случайного возраста (1-10 для собак и кошек, 1-5 для птиц)
   - Для генерации случайного веса (5-25 для собак, 2-8 для кошек, 0.1-1.1 для птиц)

3. Выведите результаты работы программы в консоль:
   - Общее количество животных каждого вида
   - Действия каждого животного (есть, спать, издавать звук, уникальное действие)

Дополнительные задания:
- Добавьте проверку, чтобы сумма всех животных всегда равнялась 20
- Реализуйте форматированный вывод для лучшей читаемости результатов
- Добавьте комментарии к коду для объяснения логики работы программы
