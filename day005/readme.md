# Задачи для тренировки

## 1. Подсчет длины строки
Напишите программу, которая принимает строку от пользователя и выводит количество символов отличных от пробелов, точек и запятых.

```text
"Привет, мир!" -> 10
"Кукушка" -> 7
". ," -> 0
```

## 2. Поиск символа в строке
Напишите программу, которая принимает строку и символ от пользователя, затем проверяет, содержится ли этот символ в строке.

```text
"Привет, мир!" 'и' -> true
"Кукушка" '!' -> false
". ," '5' -> false
```

## 3. Подсчет определенного символа
Напишите программу, которая принимает строку и символ от пользователя, затем подсчитывает, сколько раз этот символ встречается в строке.

```text
"Привет, мир!" 'и' -> 2
"Кукушка" '!' -> 0
". ," ',' -> 1
```

## 4. Вывод каждого символа строки
Напишите программу, которая принимает строку от пользователя и выводит каждый символ этой строки на новой строке.

```text
"Кукушка" ->
К
у
к
у
ш
к
а
```

## 5. Вывод строки задом наперед
Напишите программу, которая выводит введенную строку задом наперед.

```text
"Lorem" -> "meroL"
"ipsum" -> "muspi"
"tHx" -> "xHt"
```

## 6. Проверка на заглавные буквы
Напишите программу, которая принимает строку от пользователя и проверяет, содержит ли она хотя бы одну заглавную букву.  

```text
"Lorem" -> true
"ipsum" -> false
"tHx" -> true
```

## 7. Замена пробелов на подчеркивания
Напишите программу, которая принимает строку от пользователя и заменяет все пробелы в ней на символ подчеркивания '_'. Если в строке есть несколько пробелов подряд - заменить на один символ нижнего подчеркивания.

```text
"Далеко-далеко за словесными горами" -> "Далеко-далеко_за_словесными_горами"
"В стране   гласных   и согласных живут   рыбные          тексты." -> "В_стране_гласных_и_согласных_живут_рыбные_тексты."
```

# Задачи на массивы

## 8. Создание и вывод массива

Напишите программу, в которой создайте массив из n целых чисел. Заполните его случайными числами от 1 до 100, а затем выведите содержимое массива на экран. Каждый элемент должен быть напечатан с новой строки вместе с его индексом.

Пример вывода:
```text
Введите n: 5
array[0] = 42
array[1] = 15
array[2] = 98
array[3] = 7
array[4] = 63
```

## 9. Сумма элементов массива

Создайте массив из n целых чисел. Заполните его значениями, введенными с клавиатуры. Посчитайте сумму всех элементов массива и выведите ее на экран вместе с содержимым массива.

Пример вывода:
```text
Введите n: 5
Введенный массив: [5, 2, 8, 12, 3]
Сумма всех элементов: 30
```

## 10. Поиск максимального элемента

Напишите программу, которая создает массив из n вещественных чисел. Заполните массив случайными числами от 0 до 1 (используйте Random). Найдите максимальный элемент в этом массиве и выведите на экран его значение и индекс.

Пример вывода:
```text
Введите n: 8
Массив: [0.23, 0.54, 0.71, 0.15, 0.99, 0.34, 0.42, 0.87]
Максимальный элемент: 0.99
Индекс максимального элемента: 4
```

## 11. Реверс массива

Создайте массив из n строк. Заполните его названиями дней недели. Напишите код, который меняет порядок элементов массива на обратный. Выведите содержимое массива до и после реверса.

Пример вывода:
```text
Исходный массив:
[Понедельник, Вторник, Среда, Четверг, Пятница, Суббота, Воскресенье]

Массив после реверса:
[Воскресенье, Суббота, Пятница, Четверг, Среда, Вторник, Понедельник]
```

## 12. Подсчет четных чисел

Напишите программу, которая создает массив из n целых чисел. Заполните его случайными числами от -50 до 50. Подсчитайте количество четных чисел в массиве. Выведите на экран исходный массив, количество четных чисел и сами эти числа.

Пример вывода:
```text
Введите n: 15
Исходный массив: [-23, 48, -15, 2, 19, -44, 37, 0, 10, -7, 22, -36, 41, -50, 13]
Количество четных чисел: 7
Четные числа: 48, 2, -44, 0, 10, 22, -36, -50
```

# Задача: Реализация шифра Цезаря

Описание:
Шифр Цезаря - это простой метод шифрования, при котором каждая буква исходного текста заменяется другой буквой, находящейся на определенном расстоянии (сдвиге) от нее в алфавите. Например, если сдвиг равен 3, то буква 'А' будет заменена на 'Г', 'Б' на 'Д' и так далее. Шифр Цезаря назван в честь римского императора Юлия Цезаря, который использовал его для секретной переписки.

Задача состоит в том, чтобы реализовать шифр Цезаря для кириллического алфавита. Программа должна принимать исходный текст и величину сдвига, а затем выводить зашифрованный текст.

Требования:
1. Объявите переменную `text`, которая будет хранить исходный текст для шифрования.
2. Объявите переменную `shift`, которая будет хранить величину сдвига для шифрования.
3. Создайте строковую переменную `alphabet`, которая будет содержать все буквы кириллического алфавита в нижнем регистре.
4. Создайте пустую строковую переменную `cipherText`, которая будет хранить зашифрованный текст.
5. Реализуйте цикл `for`, который будет проходить по каждому символу исходного текста.
6. Внутри цикла получите текущий символ и найдите его индекс в строке `alphabet` (с учетом регистра).
7. Если символ найден в алфавите, вычислите новый индекс, применив сдвиг по модулю длины алфавита.
8. Получите новый символ по вычисленному индексу из строки `alphabet`.
9. Если исходный символ был в верхнем регистре, переведите новый символ в верхний регистр.
10. Добавьте новый символ (или исходный символ, если он не найден в алфавите) к строке `cipherText`.
11. После завершения цикла выведите исходный текст и зашифрованный текст.

Пример использования:
```
Исходный текст: Привет, мир!
Зашифрованный текст: Тулезх, плу!
```

Подсказки:
- Для получения символа из строки по индексу используйте метод `charAt()`.
- Для поиска индекса символа в строке используйте метод `indexOf()`.
- Для перевода символа в нижний регистр используйте метод `Character.toLowerCase()`.
- Для перевода символа в верхний регистр используйте метод `Character.toUpperCase()`.
- Для проверки, является ли символ буквой в верхнем регистре, используйте метод `Character.isUpperCase()`.
- Для вычисления нового индекса используйте операцию модуля (`%`), чтобы индекс не выходил за пределы длины алфавита.

Удачи в решении задачи!
