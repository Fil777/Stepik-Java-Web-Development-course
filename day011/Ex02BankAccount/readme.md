# Техническое задание на описание класса BankAccount

1. Назначение класса:
   Класс BankAccount предназначен для представления банковского счета клиента в системе управления банковскими операциями.

2. Атрибуты класса:
   1. `balance` (тип: double) - текущий баланс счета
   2. `accountNumber` (тип: String) - номер банковского счета

3. Конструктор:
   1. `BankAccount`(String accountNumber, double initialBalance)
       - Параметры:
         - accountNumber: номер создаваемого счета
         - initialBalance: начальный баланс счета
       - Функционал: инициализация нового банковского счета с заданным номером и начальным балансом

4. Методы:
   1. `getBalance()`
       - Возвращаемое значение: double
       - Назначение: получение текущего баланса счета

   2. `deposit(double amount)`
       - Параметры:
         - amount: сумма для внесения на счет
       - Назначение: пополнение баланса счета
       - Ограничения: сумма должна быть положительной

   3. `withdraw(double amount)`
       - Параметры:
         - amount: сумма для снятия со счета
       - Назначение: снятие средств со счета
       - Ограничения:
         - сумма должна быть положительной
         - сумма не должна превышать текущий баланс

   4. `getAccountNumber()`
       - Возвращаемое значение: String
       - Назначение: получение номера банковского счета

5. Дополнительно:
   1. Атрибуты balance и accountNumber должны быть приватными для обеспечения безопасности и контроля доступа
   2. Доступ к балансу и номеру счета должен осуществляться только через соответствующие методы

6. Валидация данных:
   1. При пополнении счета (метод deposit) необходимо проверять, что вносимая сумма положительна
   2. При снятии средств (метод withdraw) необходимо проверять, что:
       - запрашиваемая сумма положительна
       - на счете достаточно средств для снятия
