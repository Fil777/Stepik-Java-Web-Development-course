# Задачи для тренировки

## 1. Заполнение массива

Напишите программу, в которой создается двумерный массив размером 4x4 и заполняется числами от 1 до 16.

## 2. Вывод элементов массива

Напишите программу, в которой создается двумерный массив размером 3x3, и выводятся все его элементы на экран.

## 3. Сумма всех элементов

Напишите программу, в которой находится сумма всех элементов в заданном двумерном массиве.

## 4. Поиск максимального элемента

Напишите программу, в которой находится максимальный элемент в заданном двумерном массиве.

## 5. Вывод главной диагонали

Напишите программу, в которой выводятся элементы главной диагонали заданного двумерного массива.

Техническое задание для новичка по приведённому коду:

## Техническое задание: Игра "Монеты на столе"

### Описание игры:
Разработать консольную игру для двух игроков, где игроки по очереди забирают монеты с игрового поля, состоящего из нескольких рядов монет.

### Требования к реализации:

1. **Игровое поле:**
   - Представлено в виде двумерного массива.
   - Количество рядов: от 1 до 10.
   - В каждом ряду: от 1 до 10 монет.

2. **Игровой процесс:**
   - Два игрока ходят по очереди.
   - На каждом ходу игрок может забрать любое количество монет из одного ряда.
   - Нельзя брать монеты из разных рядов за один ход.

3. **Условие победы:**
   - Побеждает игрок, забравший последние монеты с игрового поля.

4. **Интерфейс:**
   - Консольный ввод/вывод.
   - После каждого хода выводить текущее состояние игрового поля.

5. **Ввод данных:**
   - Запрашивать у игрока номер ряда и количество монет для удаления.
   - Проверять корректность ввода (существование ряда, достаточное количество монет).

6. **Завершение игры:**
   - После окончания игры вывести сообщение о победителе.
