# Техническое задание на разработку компонентов для работы с владельцами контактов в SQLite базе данных

**Цель проекта:**
Расширение существующего приложения управления контактами с добавлением функционала работы с владельцами контактов через
SQLite базу данных.

**1. Обновление DatabaseInitializer**

- Добавить метод создания таблицы `contact_owners` в структуру базы данных
- Реализовать логику проверки существования данных в таблице
- Наполнить таблицу тестовыми данными при первичной инициализации
- Обеспечить уникальность email и username
- Использовать UUID для генерации идентификаторов
- Добавить тестовых пользователей

**Требования к тестовым данным:**

- Минимум 3 пользователя
- Уникальные email и username
- Корректные пароли

**2. Реализация ContactOwnerRepositorySQLiteImpl**

- Полная имплементация интерфейса ContactOwnerRepository
- Использование JdbcTemplate для работы с базой данных
- Реализация всех методов интерфейса:
    * `findAll()` - получение списка всех владельцев
    * `findById()` - поиск владельца по идентификатору
    * `findByEmail()` - поиск владельца по email
    * `save()` - сохранение нового или обновление существующего владельца
    * `deleteById()` - удаление владельца по идентификатору
    * `findByUsername()` - поиск владельцев по username
    * `searchByKeyword()` - поиск владельцев по ключевому слову

**Дополнительные требования:**

- Обработка возможных исключений при работе с базой данных
- Использование параметризированных запросов
- Маппинг результатов запросов на объекты модели

**Ожидаемый результат:**

- Полностью функциональный репозиторий для работы с владельцами контактов
- Надежное хранение и извлечение данных из SQLite базы
- Возможность выполнения всех CRUD операций
- Корректная работа с тестовыми данными

**3. Интеграция в текущее решение**

- Сделать реализацию **ContactOwnerRepositorySQLiteImpl** основной для приложения
- Настроить автоматическую инициализацию базы данных при старте приложения
- Обеспечить корректную работу существующих сервисов и контроллеров с новой реализацией репозитория
- Добавить необходимые Spring-конфигурации для работы с SQLite
